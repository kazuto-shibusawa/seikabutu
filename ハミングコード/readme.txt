ハミング符号と検査行列を組み合わせたものを、Cとし、静的グローバル変数として初期化してある。

ham_syn()では、キーボードから入力された受信語を引数とし、シンドロームを作成しそれを返す。

ham_che()では、キーボードから入力された受信語を引数とし、誤りが無ければ8、誤りがあるのであれば
ビットの誤りの位置である1~7の数値を返す。また、入力された文字が0と1以外の文字であれば9、
入力された文字が7ビット以外の長さであれば10を返す。
もし、想定外の処理が起きた場合は11を返し、エラーとする。

ham_inv()では、ham_che()で誤りがあるとされた位置にあるビットを反転(0→1、1←0)させ、正しい受信語をかえす。

read_line()では、入力された受信語をポインタで確保した配列に入れ、その配列を返す。
